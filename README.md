🎓 Student E-Commerce Website
A simple front-end e-commerce web application built using HTML, CSS, JavaScript, and TypeScript. This project demonstrates core concepts of UI design, product listing, and basic shopping interactions without a backend.

🚀 Features
🛍️ Product listing page
🔍 Search and filter functionality (if implemented)
🛒 Add to cart UI
📱 Responsive design for mobile and desktop
⚡ Interactive UI using JavaScript & TypeScript

🛠️ Tech Stack
HTML5 – Structure of the web pages
CSS3 – Styling and layout
JavaScript (ES6+) – Interactivity
TypeScript – Type-safe scripting
🎯 Learning Objectives

This project helped in understanding:
DOM manipulation using JavaScript
Type safety with TypeScript
Responsive UI design
Structuring front-end projects
